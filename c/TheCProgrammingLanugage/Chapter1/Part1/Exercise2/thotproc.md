Exercise:
- Experiment to find out what happens when prints's argument string contains \c, where c is some character not listed above.

## Listed escape sequences:
- \n - newline
- \t - tab
- \b - backspace
- \" - double quote 
- \\ - backslash itself 

## Exercise set
- \a - alert (bell) character - gives off a bell sound 
- \b - backspace 
- \c - unknown


# Complete set 
- \a - alert (bell) character 
- \b - backspace 
- \f - formfeed - skips to the start of the next page. used more for terminals where the output device is a printer. 
- \n - newline
- \r - carriage return - goes to the beginning of the current line and rewrites output from there. 
- \t - horizontal tab
- \v - vertical tab - used for printers to speed up vertical movement 
- \\ - backslash
- \? - question mark
- \' - single quote 
- \" - double quote 
- \NNN - octal value
- \xNN - hexadecimal value
